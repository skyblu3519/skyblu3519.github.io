JavaWeb之jar包Debug与Patch

2024-01-03 1 min read # JAVA

Debug

新建一个项目将jar包复制进来。

修改web.jar为web.zip解压缩为web目录

选择 File -> Project Structure... 将 web.jar，web目录下的BOOT-INF/classes、BOOT-INF/lib、org目录导入lib

新建一个web.jar的运行配置

右上角开启Debug，在jar包里面下断点就可以调试了

Patch

比如这里我要修复index.class里面的代码，首先新建一个与它一样的包结构com.example.jackson

写好修复后的代码重新编译

在target目录下找到编译好的Index.class

打开winrar文件，用编译好后的class文件拖进去直接替换

将替换后的zip后缀改为jar，代码成功替换
